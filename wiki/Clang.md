Clang
=====

Clang is a compiler set based on LLVM. It is distributed under BSD
Licence.

Installation
------------

Install clang form the Official Repositories.

Build packages with Clang
-------------------------

Add export CC=clang and (for c++) export CXX=clang++ to your
/etc/makepkg.conf.
